# Airline Reservation and Scheduling System
## CSC 4303 - Database Management Systems Project

This project implements an Airline Reservation and Scheduling System using a database deployed on Google Cloud.
It manages airline operations such as flight scheduling, passenger reservations, and ticket booking.

### Objectives 
* Design a relational database for airline operations
* Deploy and manage a live database using Google Cloud
* Handle passenger reservations and flight scheduling

### Features
* Flight scheduling
* Passenger reservation and booking system
* Ticket and seat assignment
* Use of primary & foreign keys, complex joins, and aggregate queries

### Prerequistes 
* Google Cloud
* MySQL Workbench

### Google Cloud Set Up
1. Upload SQL files to Cloud storage
   * Create a bucket in Google Cloud
   * Upload SQL files
2. Open Cloud SQL Studio
   * Go to Cloud Console
   * Go to Cloud SQL -> Cloud SQL Studio
3. Connect to the Database
   * Select the database
   * Open a SQL workspace (similar to MySQL Workbench)
4. Run SQL Scripts
   * Open SQL file into the editor
   * Execute commands in the console
